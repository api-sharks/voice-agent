# Components

Angular components in the application.

## AppComponent

Root component that bootstraps the app.

**File:** packages/web/src/app/app.component.ts

## VoiceFormComponent

Main voice interaction component.

Orchestrates the entire voice flow including microphone, barge-in detection, and form management.

**File:** packages/web/src/app/features/voice-form/voice-form.component.ts
